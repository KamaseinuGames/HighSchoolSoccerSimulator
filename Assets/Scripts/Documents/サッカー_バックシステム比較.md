# サッカー バックシステム比較

## 各バックシステムの特徴

### 3バック（3CB）
**代表的フォーメーション**: 3-4-3, 3-5-2

| 強み | 弱み |
|------|------|
| 中央の守備が厚い | サイドが手薄になりやすい |
| ビルドアップで数的優位を作りやすい | WBの運動量が必要 |
| 攻撃に人数をかけられる | 1対1に強いCBが3人必要 |

---

### 4バック（4DF）
**代表的フォーメーション**: 4-4-2, 4-3-3, 4-2-3-1

| 強み | 弱み |
|------|------|
| バランスが良く安定感がある | 中央突破されると脆い場合も |
| 世界的に最もスタンダード | SBの攻守のバランスが難しい |
| 選手の適応がしやすい | 相手の3トップに対して数的不利 |

---

### 5バック（5DF）
**代表的フォーメーション**: 5-3-2, 5-4-1

| 強み | 弱み |
|------|------|
| 守備が非常に堅い | 攻撃の人数が少ない |
| カウンター向き | 主導権を握りにくい |
| 強豪相手に有効 | 押し込まれると逆に危険 |

---

## フォーメーション相性

### 相性まとめ

| 守備側 | 苦手な相手 | 得意な相手 |
|--------|------------|------------|
| **3バック** | 4バックのサイド攻撃 | 中央突破してくる相手 |
| **4バック** | 3トップ・ワイド攻撃 | 守備的な相手 |
| **5バック** | ポゼッション型 | 1トップ・中央突破型 |

### 対戦相性図

```
3バック vs 4バック → 4バック有利（サイドで崩せる）
3バック vs 5バック → 3バック有利（中盤で数的優位）
4バック vs 5バック → 5バック有利（守備が安定）
```

---

## ゲーム実装への応用案

- **サイド攻撃力 vs サイド守備力**
- **中央突破力 vs 中央守備力**
- **ポゼッション vs カウンター適性**

などのパラメータで相性をシミュレート可能
